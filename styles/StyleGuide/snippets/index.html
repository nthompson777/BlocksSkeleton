Show More/Less Back to TOC

  • Make sure Mgp.js file is included in your page
  • <a href="#" id="toggle--@Model.Kind.ToString().ToLower()" title="@Model.Description Plan Summary Show More" class="moreless small toggle">Show more</a></li>
  • Initialize the Mgp.Toggle('[idNameHere]'); function in your custom script from Mgp.js
  • Repurpose the id that you used for the anchor toggle by assigning it to any element (using it as a class) that you wish to show/hide (see the "File Examples of Usage" below)

Smooth Scroll Links Back to TOC

  • Create an anchor tag with the class="page-scroll" and give the href tag a unique name - href="[unique-jump-to-name]"
  • Add the <a name="[unique-jump-to-name]"> in the location where you want the page to jump to (this Style Guide uses this Plugin)

code snippets 

  <a href="#blah" class="page-scroll">Smooth Scroll Link</a>
  
  <p> Some text here </p>
  
  <a name="blah"></a> <!-- This is where your page will scroll/jump to -->

Make sure "mgp.js" is included.
MGP location: /Scripts/Shared/Mgp.js

Tooltips & Help Modals Back to TOC

Tooltips

  • Add the Razor tag: @Html.HelpTooltipLink("[Your tooltip text here]")
  • Replace the "[Your tooltip text here]" with your custom tooltip text

MGP Tooltips are triggered by an onClick event by default unless customized (see .tooltip-alt in Mgp.js which uses onHover)
Make sure "mgp.js" is included.
MGP location: /Scripts/Shared/Mgp.js

Help Modal

  • Create the new Help Modal HTML content as a .cshtml file in /Areas/Help/Views/Explain
  • Run T4MVC Tool
  • Insert this snippet where you want the Help Modal replacing the file loc and the accessibility text:
    @Html.ExplainLink(MVC.Help.Explain.[FileLocation](), new { title = "[Accessibility Text Here]" })

code snippets 

  <!-- Tooltip Example -->
  @Html.HelpTooltipLink("See how your personal and family dynamics affect your retirement plan.")


  <!-- Help Modal Example -->
  @Html.ExplainLink(MVC.Help.Explain.PersonalReturnMethod(), new { title = "Return Method Help link" })

Validation Back to TOC

jQuery Validate Plugin with HTML5 Data Attribute Rules

Add rules to your input elements:

Example - data-rule-[rule name separate by dashes]="true"   OR   ["int value here"]
(see "code snippets" below)

Required - data-rule-required="true" (or simply use HTML5 "required")
Email - data-rule-email="true"
Minimum Length - data-rule-minlength="6"
