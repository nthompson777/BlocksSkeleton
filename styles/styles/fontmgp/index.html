Note

All icon class names must be prefixed by "fmgp" (example - class="fmgp fmgp-supersolve").

Icon size inherits font-size and color of parent element. Icon above (first example in Example Uses) is wrapped in a h2 tag.

Install As System Font

For use with desktop applicatons (Photoshop, Illustrator, PowerPoint, etc.), install font-mgp.ttf to your system font, set it as the font in your application, and copy and paste the specific icon that you want to use (not the unicode) directly from this page into your designs.
